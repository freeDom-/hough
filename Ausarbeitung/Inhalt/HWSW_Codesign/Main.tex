Das Main Modul definiert den Einstiegspunkt in das Programm. Hier wird \ac{SDL 2} initialisiert und es werden die verschiedenen Argumente ausgewertet. Falls dem Programm nicht alle möglichen Parameter übergeben wurden, werden diese mit Standardwerten initialisiert.
\\
Anschließend wird das Bild geladen und es werden nacheinander die einzelnen Module aufgerufen. Hierbei wird für jedes Modul die benötigte Laufzeit gemessen und nach erfolgreicher Ausführung ausgegeben. Außerdem wird das erzeugte Bild abgespeichert. Zuletzt werden die von der \ac{CHT} gefundenen Kreise mit Hilfe des \emph{Bresenham Algorithmus} in das von der Kantenerkennung erzeugte binäre Gradientenbild eingezeichnet.
