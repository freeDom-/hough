Das Main Modul definiert den Einstiegspunkt in das Programm. Hier wird \ac{SDL 2} initialisiert und es werden die verschiedenen Argumente, die dem Programm übergeben werden können, ausgewertet. Falls dem Programm nicht alle möglichen Parameter übergeben wurden, werden diese mit Standardwerten initialisiert.
\\
Anschließend wird das zu verarbeitende Bild geladen und es werden nacheinander die einzelnen Module aufgerufen. Hierbei wird für jedes Modul die benötigte Laufzeit gemessen und nach erfolgreicher Ausführung ausgegeben. Dabei wird das von jedem Modul erzeugte Bild abgespeichert. Zuletzt werden die von der \ac{CHT} gefundenen Kreise mit Hilfe des \emph{Bresenham Algorithmus} in das von der Kantenerkennung erzeugte binäre Gradientenbild eingezeichnet und ebenfalls abgespeichert.
